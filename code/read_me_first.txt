Το project αυτό αποτελεί την διπλωματική εργασία μου. Πρόκειται για ένα e-learning σύστημα 
για την εκμάθηση και οπτικοποίση γεωμετρικών σχημάτων το οποίο προορίζεται κυρίως για μαθητές δημοτικού. 

Για το project αυτό χρησιμοποιήθηκαν οι ακόλουθες τεχνολογίες: HTML, PHP, JAVASCRIPT, CSS, SVG και MySQL. 

Μέσω του συστήματος αυτού οι μαθητές έχουν την δυνατότητα να μελετούν θεωρητικά (κύκλος, τετράγωνο, κτλ) 
και να σχεδιάζουν τα δικά τους γεωμετρικά σχήματα. Στα γεωμετρικά σχήματα που σχεδιάζουν στην συνέχεια καλούνται 
να κάνουν ένα σύνολο ασκήσεων όπως για παράδειγμα την εύρεση του εμβαδού ή της περιμέτρου.
Επίσης οι μαθητές έχουν την δυνατότητα να εγγράφονται στη βάση δεδομένων των του συστήματος και μέσω αυτής
να ελέγχουν την πρόοδό τους. Την ίδια δυνατότητα έχει και ο δάσκαλος κάποιας τάξης της οποίας οι μαθητές 
χρησιμοποιούν το σύστημα αυτό.

Για να το τρέξετε αρκεί να ανοίξετε το αρχείο main.php και να εγγραφείτε μέσω της σελίδας Συνδεσης.
